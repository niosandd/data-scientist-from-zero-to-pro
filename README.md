## 📚 Модуль «Введение в Data Science»

Вводный модуль для тех, кто только начинает путь в Data Science.  
Курс построен по методологии **CRISP-DM** и знакомит с тремя ключевыми ролями:

| Роль | Фокус | Основные навыки |
|------|-------|-----------------|
| **Data Analyst** | Бизнес + данные | Статистика, визуализация, Excel/BI, soft skills |
| **ML Engineer** | Модели + алгоритмы | Python, математика, ML-алгоритмы, мониторинг моделей |
| **Data Engineer** | Данные + пайплайны | ETL, Big Data, инфраструктура, автоматизация |

### Основные темы модуля
1. Что такое Data Science и чем он отличается от традиционного программирования
2. Почему сейчас лучшее время изучать Data Science
3. Роли Data Analyst / ML Engineer / Data Engineer — общее и различия
4. Реальные кейсы применения ML и AI
5. Введение в Python, переменные, типы данных, условия, циклы
6. Основы алгоритмов и структур данных
